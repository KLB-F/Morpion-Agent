# Morpion-Agent

Projet d’application en apprentissage par renforcement.

#### Objectif : Implémenter et valider la méthode d’itération sur les politiques dans un cas concret.

#### Réalisation : 

Développement d’un jeu de morpion en ligne de commande intégrant une intelligence artificielle en C. L'agent étant basé sur la méthode d'itération sur les politiques.

#### Résultat : 

Obtention d'un agent convergeant rapidement (7  itérations) vers une politique stable.
